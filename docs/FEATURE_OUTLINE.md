## 2. 当前更适合答辩展示的主线

推荐按下面顺序演示：

1. 用户注册或登录
2. 查看用户资料和空间占用
3. 上传文件并刷新列表
4. 生成分享链接并通过分享链接下载
5. 输入另一位用户 ID，完成站内消息发送与短轮询刷新

这样能覆盖用户系统、文件系统、分享能力和消息模块四条主线，比较适合课程设计或毕设演示。
